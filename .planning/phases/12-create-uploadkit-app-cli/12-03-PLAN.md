    `initGit(dir)`: runs `git init`, `git add -A`, `git commit -m "chore: initial commit from create-uploadkit-app"`. Wrapped in try/catch — if git is missing, warn (`picocolors.yellow`) and continue.

    `scaffold(opts)` orchestrates, printing progress via `@clack/prompts` spinners:
    1. Resolve `templates/<template>` directory relative to the bundled package (`new URL('../templates/' + opts.template, import.meta.url)`).

    5. `writeEnvLocal(projectDir)` (if not already present).
    6. If `opts.install`: `installDeps`.
    7. If `opts.gitInit`: `initGit`.
    8. Print success block with next-steps commands (cd, run dev).
